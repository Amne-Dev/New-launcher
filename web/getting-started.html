Getting Started

Last Updated: 2026-01-20

System Requirements

  • Windows 10/11 (x64)
  • Java 17+ (bundled with installer or use system Java)
  • 200 MB free disk for launcher + additional space for Minecraft installations

Installation (Windows)

  1. Download the latest `NLCSetup.exe` from the Releases page.
  2. Run the installer and follow prompts.
  3. Launch the app from the Start Menu or desktop shortcut.

First Run

When you first run the launcher, create a profile and select an installation (Latest Release). Configure RAM and Java arguments in Settings if needed.
