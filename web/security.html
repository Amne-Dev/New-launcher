Security / Vulnerability Disclosure

Last Updated: 2026-01-20

Reporting Security Issues

To report a security vulnerability, please open a private issue or contact the maintainers directly via the contact method listed in the repository. If you must use a public issue, avoid posting exploit code or sensitive details publicly.

Responsible Disclosure

We ask that reporters follow responsible disclosure practices: provide enough detail to reproduce, avoid public disclosure until a fix is available, and allow maintainers reasonable time to respond.
